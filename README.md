# DesertFlow Plumbing Co.

Static multi-page website for a local Arizona plumbing company. Built with plain HTML, CSS, and JavaScript so it can be deployed easily with GitHub Pages.

## Assumptions used

- Business name: DesertFlow Plumbing Co.
- Location: Phoenix, Arizona
- Goal: Generate calls, quote requests, and local SEO traffic
- Style: Clean, modern, trustworthy, slightly premium

## Pages

- `index.html` — homepage
- `about.html` — company overview
- `services.html` — service directory
- `emergency-plumbing.html` — emergency plumbing page
- `water-heater-repair.html` — water heater page
- `drain-cleaning.html` — drain cleaning page
- `leak-repair.html` — leak repair page
- `service-areas.html` — service area page
- `resources.html` — helpful articles/resources page
- `contact.html` — contact and quote request page
- `privacy.html` — privacy notice

## Before launch

Replace the placeholder phone number, email address, business details, service areas, and any licensing wording with verified business information. The contact form opens a mail draft through the visitor's email app. Use a secure form service or backend if direct online submissions are needed.

## Deploy with GitHub Pages

1. Open the repository on GitHub.
2. Go to Settings > Pages.
3. Set the source to the `main` branch and root folder.
4. Save and wait for the site to publish.
